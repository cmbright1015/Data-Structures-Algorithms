Caleb Bright QuickSort Project

1. Run "make" in the Project directory
2. Run "./Bright_Caleb_InputFileGenerator to create 25 files for sizes 10, 100, 1000
3. To QuickSort single files run "./Bright_Caleb_QuickSort Bright_Caleb_input_10_XX.txt
    a. (Replace 10 with 100 or 1000 to run those files and replace XX with 01-25)
4. Run Bright_Caleb_run_all.sh to QuickSort all files and to generate the averageExecutionTime.txt file
5. The plot is included in the project but can be obtained by running Bright_Caleb_averageExecutionTime.py
